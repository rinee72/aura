# 앱 실행 최종 해결 방법

## 🔍 근본 원인

**문제**: `flutter.bat` 파일 내부에서 `WHERE /Q git` 명령어를 사용하는데, PowerShell에서 배치 파일을 직접 실행하면 이 명령어가 제대로 작동하지 않습니다.

**해결**: 배치 파일(`.bat`)은 반드시 CMD(명령 프롬프트)에서 실행해야 합니다.

## ✅ 해결 방법

### 방법 1: 배치 파일 더블클릭 (가장 간단!)

1. **파일 탐색기**에서 `C:\modu\aura_app` 폴더 열기
2. **`run_app.bat`** 파일을 **더블클릭**
3. 명령 프롬프트 창이 열리며 앱이 실행됩니다
4. Windows Desktop 앱 창이 열립니다!

### 방법 2: 명령 프롬프트(CMD)에서 직접 실행

1. **Windows 키 + R** → `cmd` 입력 → Enter
2. 다음 명령어 실행:

```cmd
cd C:\modu\aura_app
set "PATH=C:\Program Files\Git\cmd;C:\Program Files\Git\bin;%PATH%"
C:\flutter\bin\flutter.bat run -d windows --dart-define=ENVIRONMENT=development
```

### 방법 3: PowerShell에서 CMD 호출

PowerShell에서:

```powershell
cd C:\modu\aura_app
& C:\Windows\System32\cmd.exe /c "set PATH=C:\Program Files\Git\cmd;%PATH% && C:\flutter\bin\flutter.bat run -d windows --dart-define=ENVIRONMENT=development"
```

## 🎯 앱이 실행되면

1. 로그인 화면이 표시됩니다
2. 로그인 후 홈 화면에서:
   - **"전체 진행 상황 보기"** 버튼 클릭
   - 또는 AppBar의 **대시보드 아이콘** 클릭

## ⚠️ 중요 사항

- **절대 PowerShell에서 `flutter.bat`를 직접 실행하지 마세요!**
- 배치 파일은 반드시 CMD에서 실행하거나 더블클릭하여 실행하세요
- `run_app.bat` 파일이 자동으로 CMD 환경에서 실행되므로 안전합니다

## 🔧 문제가 계속되면

1. `run_app.bat` 파일이 제대로 있는지 확인
2. Git이 `C:\Program Files\Git`에 설치되어 있는지 확인
3. Flutter가 `C:\flutter`에 설치되어 있는지 확인

# Windows 개발자 모드 활성화 가이드

## 🎯 목적

Flutter 웹 앱을 실행하기 위해 Windows의 Developer Mode를 활성화합니다. 이 모드는 symlink(심볼릭 링크) 지원을 제공하여 Flutter 플러그인 빌드에 필요합니다.

---

## 📋 방법 1: 설정 앱을 통한 활성화 (가장 쉬운 방법)

### 단계별 가이드

#### 1단계: 설정 앱 열기
- **방법 A**: `Win + I` 키를 동시에 누르기
- **방법 B**: 시작 메뉴에서 "설정" 검색 후 클릭
- **방법 C**: 작업 표시줄의 시작 버튼 클릭 → 설정 아이콘 클릭

#### 2단계: 개발자용 메뉴 찾기
1. 설정 창이 열리면 왼쪽 사이드바에서 **"개인 정보 보호 및 보안"** 클릭
   - 또는 설정 창 상단의 검색창에 "개발자" 입력
2. 오른쪽 패널에서 아래로 스크롤하여 **"개발자용"** 섹션 찾기
   - 또는 직접 "개발자용" 검색

#### 3단계: 개발자 모드 활성화
1. "개발자용" 섹션에서 **"개발자 모드"** 토글 찾기
2. 토글을 **"켜기"**로 변경
   - 토글이 파란색으로 변경되면 활성화됨

#### 4단계: 확인 대화상자
- 개발자 모드를 켜면 Windows가 경고 메시지를 표시할 수 있습니다
- **"예"** 또는 **"개발자 모드 사용"** 클릭하여 확인

#### 5단계: 완료 확인
- 토글이 **"켜기"** 상태이고 파란색으로 표시되면 완료
- 별도 재부팅은 필요 없지만, 변경사항이 즉시 적용되지 않을 수 있음

## 📋 방법 2: 명령어를 통한 빠른 접근

### PowerShell 또는 명령 프롬프트 사용

1. **PowerShell 열기**:
   - `Win + X` 키를 누르고 "Windows PowerShell" 또는 "터미널" 선택
   - 또는 시작 메뉴에서 "PowerShell" 검색

2. **설정 앱 열기**:
   ```powershell
   start ms-settings:developers
   ```
   이 명령어를 입력하고 Enter 키를 누르면 개발자용 설정 페이지가 바로 열립니다.

3. **방법 1의 3단계부터 진행**

---

## 📋 방법 3: 레지스트리를 통한 활성화 (고급 사용자용)

⚠️ **주의**: 레지스트리 편집은 신중하게 진행해야 합니다. 잘못된 수정은 시스템 문제를 일으킬 수 있습니다.

### 단계

1. **레지스트리 편집기 열기**:
   - `Win + R` 키를 누르기
   - `regedit` 입력 후 Enter

2. **경로로 이동**:
   ```
   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\AppModelUnlock
   ```

3. **값 확인/생성**:
   - `AllowDevelopmentWithoutDevLicense` 값이 있는지 확인
   - 없으면: 오른쪽 클릭 → 새로 만들기 → DWORD (32비트) 값
   - 이름: `AllowDevelopmentWithoutDevLicense`
   - 값 데이터: `1` (16진수)

4. **레지스트리 편집기 닫기**

---

## ✅ 활성화 확인 방법

### 방법 1: 설정 앱에서 확인
- 설정 > 개인 정보 보호 및 보안 > 개발자용
- "개발자 모드" 토글이 "켜기" 상태인지 확인

### 방법 2: PowerShell에서 확인
```powershell
Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\AppModelUnlock" -Name "AllowDevelopmentWithoutDevLicense"
```
값이 `1`이면 활성화됨

### 방법 3: Flutter 실행으로 확인
```powershell
cd c:\modu\aura_app
flutter run -d chrome --dart-define=ENVIRONMENT=development
```
"Building with plugins requires symlink support" 오류가 사라지면 성공

## 🔧 문제 해결

### 문제 1: "개발자용" 메뉴가 보이지 않음
**해결책**:
- Windows 10 버전 1703 이상이 필요합니다
- Windows 업데이트 확인: 설정 > 업데이트 및 보안 > Windows 업데이트

### 문제 2: 토글을 켜도 적용되지 않음
**해결책**:
1. 컴퓨터 재시작
2. 관리자 권한으로 PowerShell 실행 후 다시 시도
3. Windows 업데이트 확인

### 문제 3: 권한 오류 발생
**해결책**:
- 관리자 권한이 필요할 수 있습니다
- PowerShell을 "관리자 권한으로 실행" 후 다시 시도

## 📝 활성화 후 다음 단계

1. **Flutter 웹 앱 실행**:
   ```powershell
   cd c:\modu\aura_app
   flutter run -d chrome --dart-define=ENVIRONMENT=development
   ```

2. **Chrome에서 확인**:
   - Flutter가 자동으로 Chrome을 열거나
   - `http://localhost:50000` (또는 표시된 포트) 접속

3. **컴포넌트 카탈로그 확인**:
   - AppBar의 팔레트 아이콘(🎨) 클릭
   - 또는 `/showcase` 경로로 직접 접속

## ⚠️ 주의사항

1. **보안**: 개발자 모드는 시스템 보안 설정을 완화합니다. 신뢰할 수 있는 소프트웨어만 실행하세요.

2. **성능**: 개발자 모드는 시스템 성능에 큰 영향을 주지 않습니다.

3. **비활성화**: 필요 없을 때는 토글을 "끄기"로 변경하여 비활성화할 수 있습니다.

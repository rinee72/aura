# 앱 즉시 실행 방법

## ⚠️ 현재 문제
Git PATH 문제로 Flutter가 실행되지 않고 있습니다.

## ✅ 해결 방법

### 방법 1: 환경 변수 직접 수정 (권장)

1. **Windows 키 + R** 눌러서 실행 창 열기
2. `sysdm.cpl` 입력 후 Enter
3. **"고급"** 탭 클릭
4. **"환경 변수"** 버튼 클릭
5. **"시스템 변수"** 섹션에서 **"Path"** 선택 후 **"편집"** 클릭
6. **"새로 만들기"** 클릭
7. 다음 경로 입력: `C:\Program Files\Git\cmd`
8. **"확인"** 클릭하여 모든 창 닫기
9. **컴퓨터 재시작** 또는 **로그아웃 후 다시 로그인**

### 방법 2: 새 PowerShell 창에서 직접 실행 (임시 해결)

**중요**: Cursor 외부에서 새 PowerShell 창을 열어야 합니다.

1. **Windows 키 + X** → **"Windows PowerShell"** 선택
2. 다음 명령어를 **한 줄씩** 실행:

```powershell
cd C:\modu\aura_app
$env:PATH = "$env:PATH;C:\Program Files\Git\cmd"
$env:GIT_EXEC_PATH = "C:\Program Files\Git\cmd"
C:\flutter\bin\flutter.bat run -d windows --dart-define=ENVIRONMENT=development
```

### 방법 3: 배치 파일 사용

1. 파일 탐색기에서 `C:\modu\aura_app` 폴더 열기
2. `run_app.bat` 파일을 **우클릭** → **"관리자 권한으로 실행"**

## 🎯 앱이 실행되면

1. 로그인 화면이 표시됩니다
2. 로그인 후 홈 화면에서:
   - **"전체 진행 상황 보기"** 버튼 클릭
   - 또는 AppBar의 **대시보드 아이콘** 클릭
   - 또는 주소창에 `/progress` 추가 (웹 버전인 경우)

## 💡 팁

- Windows Desktop 버전(`-d windows`)이 웹 버전보다 더 안정적으로 실행됩니다
- 환경 변수를 시스템에 영구적으로 설정하면 매번 다시 설정할 필요가 없습니다

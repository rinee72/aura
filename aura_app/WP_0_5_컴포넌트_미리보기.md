# WP-0.5: 디자인 시스템 컴포넌트 미리보기

**생성일**: 2025-12-17  
**상태**: ✅ 코드 레벨 완료 (실행은 환경 문제로 보류)

## 📱 컴포넌트 카탈로그 구성

### 1. Colors (색상 토큰)
- **Primary**: Indigo-500 (#6366F1)
- **Secondary**: Amber-500 (#F59E0B)
- **Error**: Red-500 (#EF4444)
- **Success**: Green-500 (#10B981)
- **Warning**: Orange-500 (#F97316)
- **Info**: Blue-500 (#3B82F6)
- **Text 계층**: Primary, Secondary, Tertiary, Disabled
- **Background/Surface**: Gray-50, White
- **Border/Divider/Overlay**: 다양한 회색 계열

### 2. Typography (텍스트 스타일)
- **Heading**: h1 (32px) ~ h6 (14px)
- **Body**: body1 (16px) ~ body3 (12px)
- **Label**: 14px, medium weight
- **Caption**: 12px, regular weight
- **Button**: 14px, medium weight

### 3. Spacing (간격)
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **xxl**: 48px
- **xxxl**: 64px
- **Border Radius**: 8px, 12px, 16px
- **Icon Sizes**: 16px, 20px, 24px, 32px

### 4. Buttons (버튼 컴포넌트)
- **Primary Button**: 파란색 배경, 흰색 텍스트
- **Secondary Button**: 주황색 배경, 흰색 텍스트
- **Outlined Button**: 테두리만 있는 버튼
- **Text Button**: 텍스트만 있는 버튼
- **Loading State**: 로딩 중일 때 CircularProgressIndicator 표시
- **Disabled State**: 비활성화 상태
- **Sizes**: Small, Medium, Large

### 5. Text Fields (입력 필드)
- **Label**: 필드 위에 표시되는 라벨
- **Hint**: 플레이스홀더 텍스트
- **Error Text**: 에러 메시지 표시
- **Prefix/Suffix Icons**: 아이콘 지원
- **Multi-line**: 여러 줄 입력 지원

### 6. Cards (카드 컴포넌트)
- **Basic Card**: 기본 카드 스타일
- **Tappable Card**: 탭 가능한 카드 (onTap 지원)
- **Custom Padding/Margin**: 커스텀 간격 지원

### 7. Loading (로딩 인디케이터)
- **Basic Loading**: 기본 CircularProgressIndicator
- **Loading with Message**: 메시지 포함 로딩
- **Loading Overlay**: 전체 화면 오버레이 로딩
- **Custom Size**: 크기 조절 가능

### 8. Error (에러 표시)
- **Error Widget**: 에러 메시지 표시
- **Error with Title**: 제목 포함 에러
- **Error with Icon**: 아이콘 포함 에러
- **Retry Button**: 재시도 버튼 지원
- **Error Screen**: 전체 화면 에러 표시

## 🎨 디자인 시스템 특징

### Material Design 3 기반
- `useMaterial3: true` 설정
- ColorScheme.fromSeed로 일관된 색상 생성
- 모든 컴포넌트가 Material 3 스타일 가이드 준수

### 일관성
- 모든 컴포넌트가 동일한 디자인 토큰 사용
- 색상, 타이포그래피, 간격이 통일됨
- 재사용 가능한 컴포넌트 구조

### 접근성
- 명확한 색상 대비
- 읽기 쉬운 타이포그래피
- 적절한 터치 영역 크기

## 📂 파일 구조

```
lib/
├── core/
│   └── theme/
│       ├── app_colors.dart      # 색상 토큰
│       ├── app_typography.dart # 타이포그래피 토큰
│       ├── app_spacing.dart    # 간격 토큰
│       └── app_theme.dart      # Material Theme 설정
├── shared/
│   └── widgets/
│       ├── custom_button.dart      # 버튼 컴포넌트
│       ├── custom_text_field.dart  # 입력 필드 컴포넌트
│       ├── custom_card.dart        # 카드 컴포넌트
│       ├── custom_loading.dart    # 로딩 컴포넌트
│       └── custom_error.dart      # 에러 컴포넌트
└── dev/
    └── component_showcase.dart    # 컴포넌트 카탈로그 페이지
```

## 🚀 실행 방법 (환경 문제 해결 후)

### 웹 브라우저
```bash
flutter run -d chrome
# 또는
flutter run -d edge
```

### Windows 데스크톱
```bash
flutter run -d windows
```

### 컴포넌트 카탈로그 접근
1. 앱 실행 후 AppBar의 팔레트 아이콘 클릭
2. 또는 주소창에 `/showcase` 추가

---

## ⚠️ 현재 환경 문제

1. **Windows 빌드**: CMake 빌드 에러
2. **웹 빌드**: Shader 컴파일 에러
3. **테스트 엔진**: Isolate 생성 실패

**해결 방법**:
- Flutter SDK 재설치
- Visual Studio 재부팅
- 그래픽 드라이버 업데이트
- 또는 다른 개발 환경에서 실행

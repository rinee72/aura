# AURA 앱 실행 - 최종 해결 방법

## 문제 상황

CMD 창에서 다음 오류가 발생합니다:
- `'WHERE'은(는) 내부 또는 외부 명령...이 아닙니다.`
- `Error: Unable to find git in your PATH.`

## 원인

`flutter.bat`이 내부적으로 `WHERE` 명령을 사용하여 Git을 찾는데, `WHERE` 명령이 PATH에서 찾을 수 없는 상황입니다.

## 해결 방법

### 방법 1: 수정된 배치 파일 사용 (권장)

`run_app.bat` 파일을 더블클릭하세요. 이 파일은:
- 시스템 경로를 먼저 추가하여 `WHERE` 명령이 작동하도록 함
- Git 경로를 명시적으로 설정
- `call` 명령을 사용하여 flutter.bat이 정상 작동하도록 함

### 방법 2: CMD 창에서 직접 실행

1. Windows 키 + R 누르기
2. "cmd" 입력하고 Enter
3. 다음 명령어들을 순서대로 입력:

```cmd
cd C:\modu\aura_app
set PATH=C:\Windows\System32;C:\Windows;%PATH%
set PATH=C:\Program Files\Git\cmd;%PATH%
call C:\flutter\bin\flutter.bat run -d windows --dart-define=ENVIRONMENT=development
```

**중요**: `call`을 사용해야 flutter.bat이 정상 작동합니다!

### 방법 3: PowerShell 사용

PowerShell에서는 다른 방식으로 실행:

```powershell
cd C:\modu\aura_app
$env:PATH = "C:\Windows\System32;C:\Windows;" + $env:PATH
$env:PATH += ";C:\Program Files\Git\cmd"
& C:\flutter\bin\flutter.bat run -d windows --dart-define=ENVIRONMENT=development
```

## 핵심 해결책

1. **시스템 경로를 먼저 추가**: `C:\Windows\System32`가 PATH의 앞부분에 있어야 `WHERE` 명령이 작동합니다.
2. **`call` 명령 사용**: `call C:\flutter\bin\flutter.bat` 형식으로 실행해야 배치 파일 내부 명령이 정상 작동합니다.

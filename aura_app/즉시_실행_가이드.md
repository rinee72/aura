# 앱 즉시 실행 가이드

## 현재 상황

Git PATH 문제로 Flutter가 패키지를 가져오지 못하고 있습니다. 하지만 코드는 모두 작성되어 있습니다.

## 가장 빠른 해결 방법

### 방법 1: Windows 환경 변수 직접 수정 (가장 확실)

1. **Windows 검색**에서 "환경 변수" 검색
2. **"시스템 환경 변수 편집"** 선택
3. **"환경 변수"** 버튼 클릭
4. **"시스템 변수"** 섹션에서:
   - **"Path"** 선택 → **"편집"** 클릭
   - **"새로 만들기"** 클릭
   - `C:\Program Files\Git\cmd` 추가
   - **"확인"** 클릭
5. **"시스템 변수"** 섹션에서 **"새로 만들기"** 클릭:
   - 변수 이름: `GIT_EXEC_PATH`
   - 변수 값: `C:\Program Files\Git\cmd`
   - **"확인"** 클릭
6. **모든 창 닫기**
7. **컴퓨터 재시작** 또는 **로그아웃/로그인**

### 방법 2: 관리자 권한 PowerShell에서 스크립트 실행

1. **Windows 검색**에서 "PowerShell" 검색
2. **"Windows PowerShell"** 우클릭 → **"관리자 권한으로 실행"**
3. 다음 명령 실행:

```powershell
cd c:\modu\aura_app
.\시스템_환경변수_설정.ps1
```

4. 컴퓨터 재시작 또는 로그아웃/로그인

### 방법 3: Cursor 외부에서 직접 실행

Cursor 외부에서 새 PowerShell 창을 열고:

```powershell
cd c:\modu\aura_app
$env:PATH = "$env:PATH;C:\Program Files\Git\cmd"
$env:GIT_EXEC_PATH = "C:\Program Files\Git\cmd"
C:\flutter\bin\flutter.bat run -d chrome --dart-define=ENVIRONMENT=development
```

## 앱이 실행되면

1. **홈 화면**에서 "전체 진행 상황 보기" 버튼 클릭
2. 또는 **AppBar의 대시보드 아이콘** 클릭
3. 또는 브라우저 주소창에 `/progress` 추가

# Windows 환경 변수 수정 가이드

## 문제

Flutter가 Git을 찾지 못해 앱이 실행되지 않습니다.

## 해결 방법: Windows 환경 변수 직접 수정

### 1단계: 환경 변수 창 열기

1. **Windows 키 + R** 눌러서 실행 창 열기
2. `sysdm.cpl` 입력 후 Enter
3. **"고급"** 탭 클릭
4. **"환경 변수"** 버튼 클릭

또는

1. Windows 검색에서 **"환경 변수"** 검색
2. **"시스템 환경 변수 편집"** 선택
3. **"환경 변수"** 버튼 클릭

### 2단계: 시스템 PATH에 Git 경로 추가

1. **"시스템 변수"** 섹션에서 **"Path"** 선택
2. **"편집"** 버튼 클릭
3. **"새로 만들기"** 클릭
4. 다음 경로 입력: `C:\Program Files\Git\cmd`
5. **"확인"** 클릭

### 3단계: GIT_EXEC_PATH 환경 변수 추가

1. **"시스템 변수"** 섹션에서 **"새로 만들기"** 클릭
2. 변수 이름: `GIT_EXEC_PATH`
3. 변수 값: `C:\Program Files\Git\cmd`
4. **"확인"** 클릭

### 4단계: 모든 창 닫기

모든 환경 변수 창을 닫습니다.

### 5단계: 컴퓨터 재시작 또는 로그아웃/로그인

환경 변수 변경사항을 적용하려면:
- **컴퓨터 재시작** (권장)
- 또는 **로그아웃 후 다시 로그인**

### 6단계: Cursor 재시작

1. 모든 Cursor 창 닫기
2. Cursor 다시 실행
3. 프로젝트 폴더 열기

### 7단계: 앱 실행

새 터미널에서:

```powershell
cd c:\modu\aura_app
C:\flutter\bin\flutter.bat run -d chrome --dart-define=ENVIRONMENT=development
```

## 앱이 실행되면

1. 홈 화면에서 **"전체 진행 상황 보기"** 버튼 클릭
2. 또는 AppBar의 **대시보드 아이콘** 클릭

## 참고

- 이 방법은 **가장 확실한 해결책**입니다
- 한 번 설정하면 계속 사용할 수 있습니다
- 다른 Flutter 프로젝트에서도 작동합니다

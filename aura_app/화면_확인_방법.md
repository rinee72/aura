# Flutter 앱 화면 확인 방법

## 현재 상황

다음 플랫폼에서 실행을 시도했지만 환경 문제로 실패했습니다:
- ❌ Windows: symlink 지원 문제
- ❌ Chrome/Edge: Shader 컴파일 에러
- ❌ Android 에뮬레이터: 설치되지 않음

## 해결 방법

### 방법 1: Android Studio 설치 및 에뮬레이터 사용 (권장)

1. **Android Studio 설치**
   - https://developer.android.com/studio 에서 다운로드
   - 설치 시 "Android Virtual Device (AVD)" 포함 확인

2. **AVD 생성**
   - Android Studio 실행
   - Tools > Device Manager
   - Create Device 클릭
   - 원하는 기기 선택 (예: Pixel 5)
   - 시스템 이미지 선택 (API 33 이상 권장)
   - Finish

3. **에뮬레이터 실행**
   ```bash
   cd c:\modu\aura_app
   flutter emulators --launch <에뮬레이터_이름>
   flutter run
   ```

### 방법 2: Windows 개발자 모드 재확인

1. **개발자 모드 확인**
   - Windows 설정 열기: `start ms-settings:developers`
   - "개발자용" 섹션에서 "개발자 모드" 켜기
   - 재부팅 후 다시 시도

2. **Windows 앱 실행**
   ```bash
   cd c:\modu\aura_app
   flutter run -d windows
   ```

### 방법 3: 실제 Android 기기 연결

USB 디버깅을 활성화한 Android 기기를 연결하고:
```bash
cd c:\modu\aura_app
flutter devices  # 기기 확인
flutter run
```

## 임시 해결책: HTML 미리보기

현재 환경 문제로 Flutter 앱을 실행할 수 없지만, HTML 미리보기 파일이 있습니다:
- **파일 위치**: `c:\modu\aura_app\component_preview.html`
- **열기**: 파일 탐색기에서 더블클릭하거나 브라우저로 드래그

이 파일은 실제 Flutter 앱과 동일한 디자인 토큰을 사용하여 컴포넌트를 보여줍니다.

## 다음 단계

설치 후 위의 "방법 1"을 따라 진행하시면 됩니다.

# Flutter 웹 실행 문제 해결 방법 요약

## 🔴 현재 문제

**Shader 컴파일 오류**: 한글 경로(`C:\Users\박인희\...`)에서 Flutter의 shader 컴파일러가 실패하여 웹 앱을 실행할 수 없습니다.

```
ShaderCompilerException: Shader compilation failed
```

## ✅ 해결 방법

### 방법 1: Windows Desktop 앱으로 실행 (현재 시도 중)

Windows Desktop 앱은 shader 컴파일 문제의 영향을 덜 받습니다:

```powershell
cd c:\modu\aura_app
flutter run -d windows --dart-define=ENVIRONMENT=development
```

**장점**:
- Shader 컴파일 문제 우회 가능
- 네이티브 성능
- 디자인 시스템 확인 가능

### 방법 2: 빌드 후 정적 파일 서빙

웹 빌드를 완료한 후 정적 파일을 서빙:

```powershell
cd c:\modu\aura_app
flutter build web --dart-define=ENVIRONMENT=development --release
```

빌드가 완료되면 `build\web` 폴더의 파일을 웹 서버로 서빙할 수 있습니다.

### 방법 3: Flutter SDK 경로 변경 (근본 해결)

Flutter SDK를 한글이 없는 경로로 이동:
1. Flutter SDK를 `C:\flutter`로 이동
2. 환경 변수 `PATH` 업데이트
3. `flutter doctor` 실행하여 확인

## 📝 현재 상태

Windows Desktop 앱으로 실행 중입니다. 앱이 열리면:
1. AppBar 오른쪽의 **팔레트 아이콘(🎨)** 클릭
2. 컴포넌트 카탈로그에서 디자인 시스템 확인

# Flutter Git PATH 문제 최종 해결 방법

## 문제 상황

1. ❌ `Error: Unable to find git in your PATH.`
2. ❌ PowerShell 프로세스 비정상 종료 (`-1073741819`)
3. ❌ PowerShell 인코딩 오류 (한글 경로 문제)

## 확인된 사실

✅ Git은 설치되어 있음: `C:\Program Files\Git\cmd\git.exe`
✅ PATH에도 포함되어 있음
❌ 하지만 Flutter가 Git을 찾지 못함

## 해결 방법 (우선순위 순)

### 방법 1: Flutter 설정 파일에 Git 경로 명시 (가장 확실)

Flutter는 `FLUTTER_ROOT` 환경 변수나 설정 파일을 통해 Git 경로를 찾을 수 있습니다.

**1단계: Flutter 설정 확인**
```powershell
# 새 PowerShell 창에서 (관리자 권한 불필요)
C:\flutter\bin\flutter.bat config
```

**2단계: Git 경로를 환경 변수로 설정**

Windows 환경 변수에 추가:
- 변수 이름: `GIT_EXEC_PATH`
- 변수 값: `C:\Program Files\Git\cmd`

또는 PowerShell에서 (현재 세션만):
```powershell
$env:GIT_EXEC_PATH = "C:\Program Files\Git\cmd"
```

### 방법 2: Cursor 설정 변경

**1단계: Cursor 설정 파일 수정**

`.vscode/settings.json` 파일 생성/수정:
```json
{
  "dart.flutterSdkPath": "C:\\flutter",
  "terminal.integrated.env.windows": {
    "PATH": "${env:PATH};C:\\Program Files\\Git\\cmd",
    "GIT_EXEC_PATH": "C:\\Program Files\\Git\\cmd"
  }
}
```

**2단계: Cursor 완전 재시작**

### 방법 3: Windows 환경 변수 직접 수정 (영구적 해결)

**1단계: 환경 변수 편집**
1. Windows 검색 → "환경 변수" → "시스템 환경 변수 편집"
2. "환경 변수" 버튼 클릭
3. "시스템 변수" 섹션에서 "새로 만들기" 클릭
4. 변수 이름: `GIT_EXEC_PATH`
5. 변수 값: `C:\Program Files\Git\cmd`
6. "확인" 클릭

**2단계: PATH 확인**
1. "시스템 변수"에서 "Path" 선택 → "편집"
2. `C:\Program Files\Git\cmd`가 있는지 확인
3. 없으면 추가
4. 모든 창 닫기

**3단계: 컴퓨터 재시작 (또는 로그아웃/로그인)**

### 방법 4: Flutter 캐시 정리 및 재설정

**새 PowerShell 창에서 실행:**
```powershell
cd c:\modu\aura_app

# 환경 변수 명시적 설정
$env:PATH = "$env:PATH;C:\Program Files\Git\cmd"
$env:GIT_EXEC_PATH = "C:\Program Files\Git\cmd"

# Flutter 캐시 정리
C:\flutter\bin\flutter.bat clean

# 패키지 재설치
C:\flutter\bin\flutter.bat pub get
```

### 방법 5: Git 없이 Flutter 실행 (임시 해결)

현재 `pubspec.yaml`에는 Git 의존성이 없으므로, Flutter가 Git을 찾지 못해도 작동할 수 있습니다.

**하지만 권장하지 않음**: Flutter 자체가 Git을 찾으려고 시도하므로 완전한 해결책은 아닙니다.

## 권장 해결 순서

1. **방법 3 (Windows 환경 변수 직접 수정)** ← 가장 확실
2. **방법 2 (Cursor 설정 변경)** ← Cursor 전용 해결
3. **방법 1 (Flutter 설정)** ← Flutter 전용 해결
4. **방법 4 (캐시 정리)** ← 문제 지속 시

## 확인 방법

해결 후 다음 명령으로 확인:
```powershell
# 새 PowerShell 창에서
cd c:\modu\aura_app
C:\flutter\bin\flutter.bat pub get
```

성공하면:
- `Error: Unable to find git in your PATH.` 오류가 사라짐
- 패키지가 정상적으로 다운로드됨

## 추가 참고사항

- PowerShell 인코딩 오류는 한글 경로 문제로 인한 것이지만, Flutter 코드 오류는 아닙니다
- `-1073741819` 오류 코드는 액세스 위반을 의미하며, PowerShell 프로세스가 비정상 종료된 것입니다
- Cursor를 완전히 재시작하면 환경 변수가 새로고침됩니다

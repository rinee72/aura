# AURA 앱 실행 가이드

## 빠른 실행 방법

### Windows Desktop 앱 실행
1. `run_app.bat` 파일을 더블클릭하세요.
2. 앱이 자동으로 빌드되고 실행됩니다.

### Chrome 웹 앱 실행
1. `run_app_chrome.bat` 파일을 더블클릭하세요.
2. 브라우저가 자동으로 열립니다.

## 문제 해결

### Git 오류가 발생하는 경우
배치 파일이 자동으로 Git 경로를 설정하므로 일반적으로 문제가 발생하지 않습니다.

만약 여전히 문제가 발생한다면:
1. Git이 설치되어 있는지 확인: `C:\Program Files\Git\cmd\git.exe`가 존재해야 합니다.
2. 배치 파일을 관리자 권한으로 실행해 보세요.

### 앱이 실행되지 않는 경우
1. Flutter가 설치되어 있는지 확인: `C:\flutter\bin\flutter.bat`가 존재해야 합니다.
2. 배치 파일이 있는 폴더에서 실행하세요.

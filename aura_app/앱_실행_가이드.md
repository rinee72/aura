# 앱 실행 가이드

## 현재 상황

Git PATH 문제로 인해 Flutter가 패키지를 가져오지 못하고 있습니다. 하지만 코드는 모두 작성되어 있습니다.

## 해결 방법

### 방법 1: Cursor 완전 재시작 (가장 중요!)

1. **모든 Cursor 창 닫기**
2. **작업 관리자에서 Cursor 프로세스 종료 확인**
3. **Cursor 다시 실행**
4. **프로젝트 폴더 열기**

이렇게 하면 환경 변수(`GIT_EXEC_PATH`)가 적용됩니다.

### 방법 2: 새 PowerShell 창에서 직접 실행

Cursor 외부에서 새 PowerShell 창을 열고:

```powershell
cd c:\modu\aura_app

# 환경 변수 설정
$env:PATH = "$env:PATH;C:\Program Files\Git\cmd"
$env:GIT_EXEC_PATH = "C:\Program Files\Git\cmd"

# Flutter 실행
C:\flutter\bin\flutter.bat run -d chrome --dart-define=ENVIRONMENT=development
```

### 방법 3: 컴퓨터 재시작

환경 변수가 완전히 적용되려면 컴퓨터를 재시작하거나 로그아웃/로그인해야 할 수 있습니다.

## 앱이 실행되면

1. **홈 화면**에서 "전체 진행 상황 보기" 버튼 클릭
2. 또는 **AppBar의 대시보드 아이콘** 클릭
3. 또는 브라우저 주소창에 `/progress` 추가

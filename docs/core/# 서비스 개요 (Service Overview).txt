# AURA — 셀럽·팬 소통 플랫폼 (MVP Ver.)

## 1. 서비스 한줄 정의
셀럽의 피로도를 최소화하면서 팬들과 건강하게 소통할 수 있게 돕는 "심리 안전 기반 팬소통 플랫폼".
👉 AURA는 셀럽과 팬을 보호하며 연결해주는 ‘보이지 않는 보호막’이자 ‘고유한 존재감’을 상징하는 이름

---

## 2. 서비스 목표
- 반복 질문으로 인한 셀럽의 피로도를 줄인다.
- 악성 메시지 노출을 AI로 선제적으로 차단한다.
- 팬·셀럽이 모두 안전하게 소통할 수 있는 환경을 제공한다.
- 기존 DM/댓글 구조의 문제점을 개선한 새로운 팬 경험을 만든다.

---

## 3. 핵심 문제 정의
1) 셀럽은 반복 질문, 악성 메시지로 인한 정서적 소모가 크다.  
2) 팬들은 셀럽에게 도달하기 어렵고 의미 있는 소통이 힘들다.  
3) 기존 소통 도구(V앱·버블 등)는 안전장치가 부족하거나 팬 경험이 단조롭다.

---

## 4. 핵심 기능 (MVP)
- 팬 질문 등록 (텍스트 중심)
- 질문 자동 정리 및 중복 제거(클러스터링)
- 셀럽의 답변(텍스트/짧은 음성 가능)
- 악플 자동 필터링(AI Edge Function)
- 질문/답변 목록 공개 피드
- 셀럽 관리용 대시보드(질문 정리·승인)
- 팬 프로필 & 기본 커뮤니티 기능

---

## 5. 대상 사용자
- **셀럽(연예인, 크리에이터, 인플루언서)**  
- **팬(구독자 기반 참여)**  
- **매니저(콘텐츠 정리·승인 보조)**

---

## 6. 기술 스택 (Flutter + Supabase 버전)
### 프런트엔드
- Flutter (iOS, Android, Web 동시 개발)

### 백엔드/클라우드
- Supabase Auth (로그인·권한)
- Supabase Database(PostgreSQL)
- Supabase Storage (이미지 업로드)
- Supabase Realtime (실시간 업데이트)
- Supabase Edge Functions (AI 악플 필터링)

### 배포
- 웹: Supabase Hosting  
- 앱: App Store / Google Play

---

## 7. 기대 효과
- 셀럽 소통 피로도 최소화 → 답변 효율 증가  
- 팬 경험 상승 → 서비스 재방문 증가  
- 악성 메시지 자동 차단 → 안전한 플랫폼 구축
